# Demo Scenarios

## Sentence Practice

1. 문장 연습 탭에서 기준 문장을 선택한다.
2. 사용자가 문장을 읽으면 자동 녹음이 시작되고, 침묵을 감지하면 녹음이 종료된다.
3. 로컬 Whisper 모델이 음성을 한국어 문장으로 인식한다.
4. 정답 문장과 인식 문장을 비교해 텍스트 일치도와 발음 점수를 확인한다.
5. AI 코칭 결과를 통해 발음, 속도, 문장 전달력을 확인한다.

## Conversation Practice

1. 대화 연습 탭에서 새 질문을 생성한다.
2. 사용자가 질문에 맞는 답변을 말한다.
3. 답변을 음성 인식한 뒤 문맥에 맞는 응답인지 평가한다.
4. 더 자연스러운 한국어 답변 예시를 확인한다.

## Demo Notes

- 개인 음성 데이터와 API 키는 저장소에 포함하지 않는다.
- Whisper 모델은 로컬에 따로 다운로드해야 한다.
- Gemini API 키가 없어도 기본 발음 평가는 실행할 수 있다.
